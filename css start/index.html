CSS селектори

  • # id
    Ідентифікатор id має бути унікальним і може використовуватися на сторінці лише один раз. При можливості старайтесь ходити без ідентифікаторів. Старайтесь використовувати теги або псевдокласси.
  • . class
    Cелектор класу. Відмінні селектори class від id, що клас не повинен бути унікальним і може використовуватись для вибору кількох елементів на сторінці. Використання id дозволяє вибрати конкретний унікальний елемент.
  • div
    Селектор тегу. Описує, до якого типу елементів буде застосоване CSS-правило. Селектор елемента недостатньо специфічний для того, щоб стилізувати окремі елементи, тому найчастіше застосовується для оформлення всіх тегів одного типу на сторінці.
  • li a
    Селектор нащадка або контекстний селектор застосовує стилі до дітей і нащадків елемента за будь-якої глибини вкладеності.Наприклад, замість того, щоб задати стиль усім посиланням, вам потрібно змінити відображення лише тих посилок, які розташовані всередині несортованого списку.
  • ul + p
    Це сусідній селектор. Він допоможе нам вибрати тільки той елемент, який слід відразу ж за вказаним елементом. У цьому прикладі ми вибираємо тільки перший абзац тексту, наступний відразу за тегом ul.
  • .container > ul
    Селектор прямого нащадка або дитини.Ще один поширений селектор, який дозволяє вибрати тільки безпосередню дитину всередині батьківського елементу. Селектор вибере тільки той елемент , який є прямим потомком .container > ul елемента з рівним . Він не вибере, наприклад, елемент , який є потомком першого елемента .
  • a[href] або a[href="./img.jpg"]
    Селектор атрибута. Дозволяє застосувати стилі до елементів з певним атрибутом або його значенням.
  • *
    Універсальний селектор відповідає будь-якому елементу на сторінці.
  • input[type=radio]:checked
    Даний псевдоклас вибирає тільки позначені елементи інтерфейсу користувача: радіокнопки або чекбокси .
  • ul ~ p
    Селектор вибере всі елементи p , розташовані після елемента ul на одному рівні вкладеності.
  • ‹p style="color:blue;"›
    Вбудовані (інлайн) стилі задаються тегу безпосередньо в атрибуті style. Їх неможливо масштабувати, складно перевизначати і використовувати повторно, тому вони застосовуються в дуже рідкісних випадках.
  • !important
    Правило !importantв CSS використовується, щоб додати властивості/значенню більшої важливості, ніж зазвичай.Насправді, якщо ви використовуєте !important правило, воно замінить УСІ попередні правила стилю для цієї конкретної властивості цього елемента!
